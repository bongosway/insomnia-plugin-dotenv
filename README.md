Fetch data from a .env file.

### Installation:

- Open Insomnia options (Ctrl + , | Cmd + ,)
- Type `insomnia-plugin-dotenv` on the **Plugins** tab

![](pix/install.gif)

### Usage:

- To load `.env` file via system dialog

![](pix/single.gif)

- To add `.env` via environment variable

![](pix/change.gif)
